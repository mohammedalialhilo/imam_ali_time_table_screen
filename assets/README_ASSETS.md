# Asset Notes

This project expects these logo files:

- `assets/logo-teal.png`
- `assets/logo-red.png`

Placeholder PNGs are currently included so the UI references resolve cleanly.

## Replace with official logos

1. Export the official teal logo as `assets/logo-teal.png`
2. Export the official Muharram logo as `assets/logo-red.png`
3. Keep the filenames the same so `index.html` and the theme logic continue working

## If a logo is missing

The display UI still works because it falls back to a text mark when an image fails to load.
